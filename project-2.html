Finnsmouth - Indie Game Project

A passion project of mine; a 2.5D game made in Unity with a focus on fishing

Finnsmouth Overview

Finnsmith represents everything I love about game development - a chance to merge my pixel art craftsmanship with 3D environmental modeling through one of gaming's most satisfying mechanics: fishing. This passion project blends cozy town-building progression with unsettling Lovecraftian undertones in a distinctive 2.5D Unity environment.

One particularly stubborn technical challenge early on was creating a performance-friendly water interaction system. The solution? I re-engineered the ocean shader to handle buoyancy calculations through an on-demand function call rather than continuous CPU processing. By feeding object coordinates into a modified (water equation name) calculation, the system now efficiently determines vertical positioning only when needed - keeping those fishing bobbers perfectly afloat without tanking performance.
